Extensions

Replacing MODs, Extensions are a new contribution type that require no file edits for installation. They can be easily installed via the ACP. Self-contained and provide functionality for template and php events/listerners, adding crons, providing search & capatcha plugins, template/css/language files inclusion and modules. Easily enabled, disabled (without removing db content) and purged (removes db content) in the ACP. Metadata is supported in composer.json which allows specifying phpBB versions as a constraint instead of a specific version. Custom code can be run on enabling, disabling and purging. There is also version check functionality built-in.

Symfony Components

Symfony is a well documented and adopted framework in the PHP world being developed by SensioLabs. It is used by several popular projects including Drupal & Laravel. phpBB is now also built using Symfony.

Routing & Controllers

phpBB is now being generated using Routes & Controllers in order to provide better API calls & adhere to a more modern MVC construct.

Migrations

Ajax

Traditionally with phpBB, even simple operations like deleting a post or reordering forums have involved multiple requests and full page loads, which was slow and heavy on bandwidth. We've added AJAX functionality to a lot of these simple operations, making them faster and easier to do.

Improved Style Inheritance

The tempting system has been completely overhauled. Cleaning up the folder structure and providing an increased ability to inherit from the other styles. No more unnecessary an confusing configuration files!

Improved Navigation

Redesigned and layedout to provide better UX/UI. The addition of menus to group similar functionality. Improved consistency across the design. Completely buttons leveraging CSS rather than images.

HTML5 & CSS3

Were now using the HTML5 doctype and CSS3 to modernize the codebase, reduce the number of requests made to the server, and speeding up page load times.
