Extensions

Replacing MODs, Extensions are a new contribution type that require no file edits for installation. They can be easily installed via the ACP. Self-contained and provide functionality for template and php events/listerners, adding crons, providing search & capatcha plugins, template/css/language files inclusion and modules. Easily enabled, disabled (without removing db content) and purged (removes db content) in the ACP. Metadata is supported in composer.json which allows specifying phpBB versions as a constraint instead of a specific version. Custom code can be run on enabling, disabling and purging. There is also version check functionality built-in.

Ajax

Traditionally with phpBB, even simple operations like deleting a post or reordering forums have involved multiple requests and full page loads, which was slow and heavy on bandwidth. We've added AJAX functionality to a lot of these simple operations, making them faster and easier to do.

HTML5 & CSS3 Enhancements

We've updated phpBB from XHTML 1.0 to HTML5, and have modernised a lot of our CSS—for example, we're now using CSS3 border radius for rounded corners instead of having images. This reduces the number of requests made to the server, and thus speeds up page load times.
